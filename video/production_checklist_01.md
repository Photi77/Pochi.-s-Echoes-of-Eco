# 制作チェックリスト：動画No.1 概要篇

## 撮影前準備

### ワールド設定
- [ ] Creative専用ワールドを新規作成（サバイバルプレイとは別ワールド）
- [ ] シード値を記録しておく（撮り直し時のため）
- [ ] ゲーム難易度：ピースフル（戦闘干渉なし）
- [ ] 時間固定コマンドを確認：`/gamerule doDaylightCycle false`
- [ ] 天候固定コマンドを確認：`/weather clear`

### MOD構成（撮影環境）
- [ ] Echoes of Eco（本体）
- [ ] Distant Horizons（描画距離 512チャンク以上推奨）
- [ ] Complementary Shaders（または同等のシェーダー）
- [ ] AmbientSounds（環境音）
- [ ] パーティクルMOD（霧表現：張家界シーン用）
- [ ] OptiFine または Iris（シェーダー互換）

### 撮影設定
- [ ] FOV：70〜80（広角気味）
- [ ] UI非表示：F1キーで確認
- [ ] ゲームバー・タスクバー非表示（録画前に確認）
- [ ] 録画解像度：1080p以上（1440p推奨）
- [ ] フレームレート：60fps固定（DHが重い場合は30fps）

## シーン別撮影チェック

### シーン1：オープニング（夜明け）
- [ ] DH 512チャンク描画で大陸全体が見える場所を探す
- [ ] 複数バイオームの境界が見えるアングル
- [ ] `/time set 22500`（夜明け前）で時間固定
- [ ] カメラプルバックを30秒程度収録
- [ ] テロップ用に「空が広い構図」も1テイク録る

### シーン2：世界の広さ（大陸俯瞰）
- [ ] カットA：山頂からの地平線（同一バイオームが続く）
- [ ] カットB：512チャンク描画でバイオーム境界線を俯瞰
- [ ] カットC：ジャイアンツコーズウェイ海岸の真上俯瞰（スペクテイターモード使用）
- [ ] 各カット10〜15秒以上収録（編集で短くできるが長い方が安全）

### シーン3A：カッパドキア
- [ ] 岩柱の「間を抜ける」移動ショット
- [ ] 夕陽が岩柱に当たる角度（`/time set 12000`前後で調整）
- [ ] 上空からの俯瞰（バイオームの広がりが分かる構図）
- [ ] テロップ用に「地平線まで岩柱が続く」構図

### シーン3B：パムッカレ
- [ ] 白いテラス状地形の真上俯瞰
- [ ] 青〜白のグラデーションが映える水面
- [ ] 地上レベルで歩くシーン

### シーン3C：張家界
- [ ] 霧パーティクルが出ている状態で石柱群
- [ ] 上空から俯瞰 → 地上に降りるカット
- [ ] 石柱の間を通り抜けるシーン

### シーン3D：ガイアナ高地
- [ ] テーブル状山頂の端から下を見る（絶壁感）
- [ ] 雲が山頂より低い構図（スペクテイターで高度調整）
- [ ] 頂上の広がりを示す横移動ショット

### シーン3E：ジャイアンツコーズウェイ
- [ ] シェーダーの水面を活かした海岸ショット
- [ ] 波が打ち寄せる場面（動きのある映像）
- [ ] 六角柱状の岩が見える地上アングル

### シーン3F：テーブルマウンテン
- [ ] 断崖絶壁の頂上から下を見る
- [ ] 雲海が下に広がる構図（雲の高さを調整）
- [ ] 頂上が平らなことが分かるアングル

### シーン3G：クリスタルケーブ
- [ ] 洞窟内の宝石ブロックが発光している場面
- [ ] カメラがゆっくり進む（前進移動）シーン
- [ ] 広い空間から宝石が見える構図

### シーン4A：ハーピーイーグル
- [ ] 張家界またはガイアナ高地バイオームで飛行シーン
- [ ] 翼のアニメーションが見えるアップ
- [ ] 木の上に止まっている場面（補助）

### シーン4B：海岸・水中生物
- [ ] アシカが海岸にいる場面
- [ ] 水中に潜るシーン（シェーダーの水中表現）
- [ ] マンティスシュリンプのアップ（動いている場面）
- [ ] ハンマーヘッドシャークの遊泳

### シーン4C：草原動物
- [ ] カピバラの群れが移動する遠景
- [ ] カピバラアップ（補助）
- [ ] キリンの全身が映るショット

### シーン4D：クリオネ召喚
- [ ] クリオネスタッフを使う場面（アイテムを持って右クリック）
- [ ] クリオネが浮遊している幻想的な場面
- [ ] 洞窟内かクリスタルケーブが背景だと映える

### シーン4E：昆虫
- [ ] トンボの羽ばたきアップ（GeckoLibアニメーション）
- [ ] チョウが飛んでいる場面
- [ ] フラワーマンティスのアップ

### シーン5A：農業（畑）
- [ ] 複数種の農作物が並んだ畑の俯瞰
- [ ] カラフルな構図にする（緑・赤・白など混在）
- [ ] 個別の作物アップ（バジル・唐辛子・アスパラ等）

### シーン5B：料理（フライパンブロック）
- [ ] ガパオライス完成シーン
- [ ] ペペロンチーノ完成シーン
- [ ] マーボーナス完成シーン
- [ ] 生姜焼き完成シーン
- ※ 各料理を順番に並べたテーブルショットもあると◎

### シーン5C：醸造システム
- [ ] 醸造設備全体の映え構図（蒸留器・発酵槽等）
- [ ] 液体8種が確認できるショット
- [ ] 製造工程の一連の流れ（短く）

### シーン6：素材・装備
- [ ] 鉱石ブロックを並べた比較ショット（クロマイト・バナジウム・チタン・マグネシウム）
- [ ] 合金インゴットの比較
- [ ] 防具セットのスライドショー（12セット以上、各3〜4秒）
- [ ] ミネラル防具の染色バリエーション（複数色を並べる）

### シーン7：マイクラらしい要素
- [ ] グラップリングフックで崖を登るシーン
- [ ] フックショットの使用シーン
- [ ] 飛行ボートで大陸を移動するシーン（DH描画で広大さを強調）
- [ ] ワニの顎チェーンまたはクリオネスタッフの使用
- [ ] 捕獲ネットで動物を捕まえるシーン（ショートとの接続用）

### シーン8：エンディング
- [ ] 夕暮れ〜星空の移行（`/time set 12000` → `/time set 18000`等で調整）
- [ ] DHで大陸全体を見渡すショット（オープニングと対をなす構図）
- [ ] エンドカード用の静止背景（15〜20秒の静的な映像）
